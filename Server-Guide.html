So what is JDEV PVP Sandbox?

The JDEV Garry’s Mod server is dedicated to a deathmatch-like experience while still allowing buildings to flourish. This encourages active PvP, along with base wars and building. You get points for each kill, allowing you to buy restricted weapons, case keys, cosmetics, and even ranks/donor vouchers. However, our server doesn’t force you to fight to the death, so you can still join and build without being bothered in the provided PvP free building zones.

Before continuing, if you haven't, you should read our server rules

Where are the build zones?

All spawn zones are automatically restricted from PVP.

Our server also features killstreaks, meaning if you survive for long periods of time, racking up kills, you will gain points at an exponential rate.
